Condicionais em Python

1. O que são estruturas condicionais?

Estruturas condicionais são usadas para que o programa tome decisões. Elas executam diferentes blocos de código dependendo se uma condição é verdadeira ou falsa. Por exemplo: se a pessoa tem 18 anos ou mais, pode votar; caso contrário, não pode.

2. Como usar if, elif e else?

  • if → se uma determinada condição for satisfeita, executa o bloco de código.
  • else → se a condição do if não for satisfeita, executa este bloco de código.

Exemplo:


idade = 19

if idade >= 18:
    print("Você é maior de idade.")
else:
    print("Você é menor de idade.")

No código acima, a variável idade recebe o valor 19. O if verifica se a idade é maior ou igual a 18. Como a condição é verdadeira, será exibida a mensagem:

Você é maior de idade.

Se a condição fosse falsa, o bloco do else seria executado:

Você é menor de idade.

Existe ainda o elif, que significa "else if". Ele é usado para verificar múltiplas condições. Caso a primeira condição (if) seja falsa, o programa testa as condições dos elif seguintes.

Se nenhuma das condições for verdadeira, o else será executado.

Exemplo:


idade = 17

if idade >= 18:
    print("Você é maior de idade.")
elif idade >= 16:
    print("Você pode votar, mas não é maior de idade.")
else:
    print("Você ainda não pode votar.")

3. Condicional Aninhada

Em Python, você pode colocar um if dentro de outro. Isso é chamado de condicional aninhada. É útil quando você precisa verificar uma condição dentro de outra.

Exemplo:

idade = 20
    tem_carteira = True

    if idade >= 18:
        if tem_carteira:
            print("Pode dirigir.")
        else:
            print("Precisa tirar a carteira.")
    else:
        print("Menor de idade.")

Como funciona:

  • Se a pessoa tem 18 anos ou mais, o programa verifica se ela tem carteira de motorista.
  • Se ela tiver, pode dirigir. Se não tiver, precisa tirar a carteira.
  • Se tiver menos de 18, o programa nem checa a carteira — já mostra que é menor de idade.

4. Operadores usados em estruturas condicionais

Para que o if funcione corretamente, é necessário avaliar uma condição. Essas condições normalmente comparam valores ou verificam estados, e para isso usamos operadores de comparação e operadores lógicos.

Operadores de comparação

Operador Significado Exemplo
== Igual a idade == 18
!= Diferente de nota != 10
> Maior que salario > 2000
< Menor que idade < 18
>= Maior ou igual a nota >= 7
<= Menor ou igual a idade <= 12

Operadores lógicos

Operador Nome Uso
and E lógico Todas as condições precisam ser verdadeiras
or OU lógico Pelo menos uma condição precisa ser verdadeira
not Negação Inverte o valor lógico (True vira False e vice-versa)

Exemplo com operadores lógicos:


    idade = 20
    tem_carteira = True

    if idade >= 18 and tem_carteira:
        print("Pode dirigir")
    else:
        print("Não pode dirigir")

5. Boas práticas

  • Use indentação correta (Python exige isso!)
  • Deixe as condições claras e objetivas
  • Evite usar elif em excesso — use else para cobrir os casos restantes
